локальный репозиторий - папки .git внутри нашего ноута
удаленный репозиторий - папки на сервере (которую мы создали в GitHub)

git init - создает git репозиторий(локальный) из текущей папки

git add . - добавляет файлы и папки для отслеживания (эту команду пишем каждый раз перед командой commit(т.е. перед сохранением)).
git commit -m 'сообщение' - создает новую версию либо же (сохраняет изменения), если были изменения

git status - показывает состояние файла.
git branch - показывает все ветки
git branch (название ветки) - создает новую ветку
git push origin master - попасть в удаленную ветку
git commit - сохраняет и создает новую версию папку или файл 
git remote add origin {ssh|https} - создает связь между локальным или удаленным репозиторием.
git check out - дома узнать
